# 🧪 Zombie Plus – Automação de Testes com Playwright

## 🤘 Sobre o projeto

O **Zombie Plus** é um projeto pessoal de automação de testes em desenvolvimento, criado com o objetivo de praticar e aplicar boas práticas de **QA e testes end-to-end (E2E)** utilizando o **Playwright**.  
A ideia é validar fluxos principais do sistema e garantir a confiabilidade da aplicação a cada nova atualização.

O projeto está sendo desenvolvido de forma incremental, evoluindo com novos cenários, melhorias na estrutura e relatórios de execução.

---

## 💻 Tecnologias utilizadas

- **Node.js**  
- **Playwright**  
- **JavaScript (ES6+)**  
- **Faker.js**  
- **PostgreSQL**  

---

## 🚀 Como executar

1. **Clonar o repositório**  
   ```bash
   git clone https://github.com/Pxtrickvieira/zombieplus-playwright.git
   ```

2. **Instalar dependências**  
   ```bash
   npm install
   ```

3. **Executar os testes**  
   ```bash
   npx playwright test
   ```

4. **Exibir o relatório de execução**  
   ```bash
   npx playwright show-report
   ```
